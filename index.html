Cell-line-recognition

Cell line recognition and normalization system, supporting corpora and tagged documents.

Welcome to Turku cell line name recognition and normalization

The project is a result of collaboration between university of Turku and Gent university. The aim of the project is to create corpora that is suitable for machine learning system to recognize establish cell line names from text. The recognized cell line names are normalized linking the names to Cellosaurus identifier.

Gellus and CLL Corpora

We created two manually annotated corpora, Gellus and CELL. Gellus is suitable for the training any machine learning system in recognizing cell line name mentions. CLL is suitable for evaluating the system in recognizing the Cellosaurus cell line names.

Tagged and normalized cell line name documents

We tagged and normalized cell line names in ~6.7 abstracts + full-text documents in EVEX database. About ~500,000 documents contain at least 1 cell line name. We provided the documents with at least one cell line name tagged in standoff format.
